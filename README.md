# wedding


## Apps Script

### prerequisite

- Google Apps Script project
- Nodejs
- [clasp](https://github.com/google/clasp)

Install clasp
```sh
npm install -g @google/clasp
```

Clone the existed project
```sh
clasp clone <scriptId> --rootDir ./appsScript
```
### Development

Pull the lasted change
```
clasp pull
```

Push your change
```
clasp push
```
